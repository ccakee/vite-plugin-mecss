# vite-plugin-mecss
# 使用

```html

<div style="width:100px;height:100px;margin:30px;color:red;border: 10px solid #ccc;">
    test text
</div>
  ||
  ||
<div w="100" h="100" m="30" color="red" bor="10 #ccc">
    test text
</div>

```
